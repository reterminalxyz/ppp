# Как добавить свои фотографии в проект

Так как Google Drive блокирует прямые ссылки на изображения (ошибка CORS / 403 Forbidden), мы перевели проект на **локальные изображения**. Это значит, что картинки должны лежать прямо внутри папки с кодом сайта.

Это сделает сайт в 10 раз быстрее и надежнее!

## Что нужно сделать перед деплоем на Render:

1. В корне вашего репозитория на GitHub (там же, где лежат файлы `package.json`, `App.tsx` и т.д.) создайте папку с названием `public`.
2. Внутри папки `public` создайте папку `images`.
3. Загрузите все ваши фотографии из Google Drive в эту папку `public/images/`.
4. **ОЧЕНЬ ВАЖНО:** Переименуйте ваши фотографии в точности так, как они прописаны в файле `data.ts`.

### Как это работает в Vite:
При сборке проекта (когда Render запускает `npm run build`), сборщик Vite автоматически берет всё содержимое папки `public` и кладет его в корень готового сайта. Поэтому в коде мы пишем путь `/images/1-opuntia-1.jpg`, и браузер легко находит эту картинку.

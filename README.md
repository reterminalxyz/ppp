# Как скачать и запустить сайт

Так как я ИИ и общаюсь с вами через интерфейс, выдавая код, у вас есть несколько способов сохранить этот сайт к себе:

## Способ 1: Скачать из песочницы (Sandbox / Agent Platform)
Если вы используете онлайн-редактор или платформу (судя по скриншотам, это Google Cloud Agent Platform / Sandbox), найдите кнопку **Export to ZIP**, **Download** или иконку скачивания (обычно выглядит как облако со стрелкой вниз или находится в меню файлов слева/справа сверху).

## Способ 2: Вручную (Локально на компьютере)
1. Создайте пустую папку на вашем компьютере (например, `pianeta-piante`).
2. Внутри создайте папку `components`.
3. Скопируйте код каждого файла из нашего чата (или из редактора слева) и сохраните их с правильными именами и расширениями:
   - `index.html`
   - `index.tsx`
   - `App.tsx`
   - `data.ts`
   - `types.ts`
   - `metadata.json`
   - `components/InteractiveHero.tsx`
   - `components/ProductCard.tsx`
   - `components/Cart.tsx`
   - `components/Menu.tsx`
   - `components/Gallery.tsx`
4. Так как сайт использует `importmap` (ESM модули загружаются прямо в браузере через CDN), вам **не нужен** сложный билд-процесс (Webpack, npm install и т.д.).
5. Чтобы сайт заработал, его нужно открыть через локальный веб-сервер (иначе браузер заблокирует импорты из-за CORS). 
   - Если у вас установлен **Python**, откройте терминал в папке с проектом и введите: `python3 -m http.server`
   - Если у вас установлен **Node.js**, введите: `npx serve`
   - Либо используйте расширение **Live Server** для VS Code.
6. Откройте предложенную ссылку (обычно `http://localhost:8000` или `http://localhost:3000`) в браузере.

## Способ 3: Запросить сборку в один файл
Если вам сложно работать с множеством React-файлов локально, вы можете написать мне: *"Собери весь сайт в один index.html файл"*. Я перепишу всю логику на чистый HTML/CSS/JS (без React), и вы сможете просто скачать один файл `index.html` и открывать его двойным кликом. (Однако текущая React-версия более масштабируемая).
